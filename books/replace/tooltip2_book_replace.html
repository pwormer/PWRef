• If an author or editor must be removed, then blank the first- and last name input field that specify the author or editor to be deleted.
  • If an author or editor must be added, hit the corresponding button. This gives a new first- and last name field that can be filled out.
  • If the author or editor list must be reordered, then add an extra author or editor field by hitting the appropriate button. Move by cut and paste (under Windows: Ctrl+X and Ctrl+V) the appropriate author/editor name to it. Repeat until the order is correct.
