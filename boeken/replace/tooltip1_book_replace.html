Usage

  1. The user fills out one or more search fields on the present page. Upon hitting the search button, a list is shown of all references that satisfy the input search parameters.
  2. The user picks one reference from this list.
  3. An edit screen is shown with all the database properties of the chosen reference.
  4. The user edits the fields and adds or deletes authors and editors.
  5. The edited information is previewed and the user is asked if it must be stored. If so, it is stored as a new database record.
  6. Finally the user is asked whether the original record must be deleted. If not, the original record is retained together with its new version.

Notes

  • All input fields are optional. The default time interval is from last year to the present year.
  • All input fields (except the years) may contain a substring of the property searched for.
  • If the default time interval is the only input, all book-type publications of this and last year will be returned.
  • Because on input the the default year is 1 CE, it should not be forgotten that publications with that year may be present in the database. These publications are to be considerd as undated.
  • The specified name may refer to any of the coauthors or coeditors of the work searched for, he/she is not necessarily the first author or editor.
  • The title field refers to a primary title (title of book or contribution etc.) and may be a substring of the title.
  • The key field is required in BibTeX and gives a unique identification of the work.
  • The type of work (book, PhD thesis, etc.) may be "All types", in which case all book-type publications are searched for.
